# 🚀 Чебоко - Система управления заявками

B2B сервис для управления заявками на обслуживание кофейных аппаратов.

## ⚡ Быстрый старт

```bash
# Запуск dev-сервера
dev.bat

# Открыть в браузере
http://localhost:3000
```

## ✅ Что реализовано

### Основное
- ✅ Авторизация (NextAuth + JWT)
- ✅ Профиль (редактирование + смена пароля)
- ✅ Заявки (создание, просмотр, смена статусов, **фото**)
- ✅ Уведомления (БД + API + UI + polling + Telegram)
- ✅ Настройки (4 вкладки с БД)
- ✅ **Действия в таблицах** (меню с быстрыми операциями)

### Действия в таблицах

#### Заявки (`/tickets`)
- 👁️ **Просмотр** - переход к деталям заявки
- ✅ **Завершить** - быстрая смена статуса на COMPLETED
- ❌ **Отменить** - быстрая смена статуса на CANCELED
- 🗑️ **Удалить** - удаление заявки (только Admin/Operator)

#### Оборудование (`/equipment`)
- ✏️ **Редактировать** - изменение параметров
- 🗑️ **Удалить** - удаление оборудования (только Admin/Operator)

#### Клиенты (`/clients`)
- ✏️ **Редактировать** - изменение данных локации
- ⋮ **Ещё** - дополнительные опции

#### Команда (`/team`)
- ⋮ **Ещё** - управление пользователем

## 🛠 Технологии

- **Frontend:** Next.js 16, React 19, TypeScript
- **Backend:** Next.js API Routes (Supabase)
- **База данных:** PostgreSQL (Supabase)
- **Авторизация:** NextAuth.js + JWT
- **Карты:** Яндекс.Карты API (ключ: e1a186ee-6741-4e3f-b7f4-438ed8c61c4b)
- **Уведомления:** Telegram Bot API (@cooolcaree_bot)
- **Хранение фото:** /public/uploads/

## 📝 Примечания

### Фото в заявках
- Загружаются в `/public/uploads/`
- Сохраняются в БД как JSON массив
- Отображаются в деталях заявки
- При ошибке - заглушка "Нет фото"

### Модальные окна
- Открываются **без затемнения** фона
- Нет размытия
- Страница остается видимой

### Карта
- Город: **Ростов-на-Дону**
- API ключ уже настроен
- Клик по метке инженера показывает информацию:
  - Район
  - Статус
  - Количество активных заявок
  - Email

### Telegram-бот
- Токен: уже в `.env`
- Chat ID: 5218294385
- Отправляет уведомления при:
  - Создании заявки
  - Завершении заявки

### Действия в таблицах
- Кнопка **⋮** открывает меню действий
- Доступные действия зависят от роли
- Меню закрывается по клику вне его
