http://localhost:3000

## 📊 Что реализовано

✅ **Авторизация** — NextAuth + JWT  
✅ **Профиль** — редактирование + смена пароля  
✅ **Заявки** — создание, просмотр, смена статусов  
✅ **Уведомления** — БД + API + UI + polling (30 сек)  
✅ **Telegram** — интеграция с ботом  
✅ **Настройки** — 4 вкладки с сохранением в БД  
✅ **Клиенты/Оборудование** — полный CRUD  
✅ **Команда** — добавление сотрудников  
✅ **Карта** — Яндекс.Карты с метками инженеров  
✅ **Ролевая модель** — CLIENT_MANAGER видит только свои заявки

## 🛠 Технологии

- **Frontend:** Next.js 16, React 19, TypeScript
- **Backend:** Next.js API Routes (Supabase)
- **База данных:** PostgreSQL (Supabase)
- **Авторизация:** NextAuth.js + JWT
- **Карты:** Яндекс.Карты API
- **Уведомления:** Telegram Bot API

## 📝 Примечания

### Медленная загрузка?
1. Проверьте подключение к Supabase
2. Для локальной разработки можно использовать SQLite
3. Отключите неиспользуемые функции в настройках

### Карта не работает?
- Получите бесплатный API ключ Яндекс.Карт
- Без ключа карта загружается в ограниченном режиме

### Уведомления не приходят?
- Проверьте Telegram бота (токен в .env)
- Убедитесь что chat_id правильный
- Уведомления создаются при смене статуса заявки
